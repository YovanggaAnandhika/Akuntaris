## Layanan

* 🔭 **Sistem Kasir.** Melakukan Pendaftaran Management Pendaftaran Mahasiswa Baru.
* 🌱 **Sistem Arus Kas Priodik.** Melakukan Metode Pembayaran Untuk Segala Macam Pendaftaran
* 👯 **Managemen Penggajian.** Menampung Big Data Mahasiswa, Termasuk Nilai, Rapor, Kurikulum, dan Feature CCTV
* 🤔 **Managemen Pesanan.** Database Guru, Sistem Belajar Mengajar, Jadwal Mengajar, dan Penilaian
* 😄 **Data Pembelian Dan Penjualan.** Orang Tua Bisa Memantau Dalam Peserta Didik Kampus
* ⚡ **Lebih Banyak Fitur Lainnya.** 🐈
